Unity版本 2018<br/>
fgui 3.9以上<br/>
leaf go 稳定最新版<br/> 
protobuf使用2.0版本<br/>
目前能调通<br/>
该项目主要整合 luaFrameWork(ugui)   fgui框架 <br/>
服务端采用 leaf（go） 作为服务端框架 <br/>
通信采用protobuf  lua层使用pb_lua 服务端请使用proto进行通信
客户端通过 LuaNetWork\Assets\LuaFramework\Scenes\main 进行启动，运行或打包请把Scenes全部添加到build setting中
